採点手順

  1. 採点者IDを入力します。
  2. 採点する課題と参加者IDを選びます。
  3. 採点キューを準備し、採点を開始します。
  4. 各音声を聞き、正誤と発話開始位置を確認します。
  5. 必要に応じて波形上でonset / offsetを修正します。
  6. 最後にCSVまたはJSONを書き出します。

ショートカット

Space

M

R
onsetの少し前から再生
0 / 5 / 1
正確性を0 / 0.5 / 1に設定
9
NRに設定
C
自動onsetを確認済みにする
← / →
